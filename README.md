error.base is a debian meta package that contains..
errpapers ( error.os core wallpapers )
sddm-theme-error ( error.os kde breeze forked sddm theme )
The whole monochrome  kde theme for all users not only for one user ( credits pwde i just added it as a debian package )
theme-icons-kora ( credits to bikaas for his wonderful  creation )
theme-cursor-bibata ( this version is minimal with only 2 cursor theme )
plymouth-theme-error
fonts-kawkab-mono ( only truetype )
a font from this website [ekushey](https://okkhor52.com/download.html?id=_024) 

so in conclusion this whole .deb package contains so many stuffs to turn a ugly graphicless minimal kde system to look good
